## Super Quiz Master (CompTIA Practice)

> _Practice quizzes for CompTIA A+ & Security+ certification exams._

- **Features:**
  - Dynamic quiz engine with question bank
  - User-generated questions
  - Real-time feedback and scoring
  - Professional color palette and typography
  - Built with jQuery, Bootstrap, Font Awesome
- **Directories:**
  - `super-quiz-master/compTIA/`
  - `super-quiz-master/testQuiz/`
- **Micro-interaction:** Animated medal icons, progress bars
